Feeds

  • very powerfull module (framework) for importing 3rd party data into Drupal
  • plugabble, flexible
  • uses abstract workflow that can cover wide set of data formats
  • can import XML, HTML, RSS, CSV, OPML, ... (you name it!)
  • data can be saved as nodes, terms, users, DrupalCommerce products, ...

http://drupal.org/project/feeds
